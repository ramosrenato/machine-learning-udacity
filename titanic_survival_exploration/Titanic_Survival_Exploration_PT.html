def accuracy_score(truth, pred):
    """ Devolver a medida da preciso para um conjunto de valores reais e estimativas. """
    
    # Certificar-se de que h o mesmo nmero de valores reais e estimativas
    if len(truth) == len(pred): 
        
        # Calcular e devolver a preciso como uma porcentagem
        return "Estimativas tm uma preciso de {:.2f}%.".format((truth == pred).mean()*100)
    
    else:
        return "Nmero de estimativas no bate com o nmero de resultados!"
    
# Testar a funo 'accuracy_score'

def predictions_2(data):
    """ Modelo com dois atributos: 
            - Estimar a sobrevivncia do passageiro se ele for do gnero feminino.
            - Estimar a sobrevivncia do passageiro se ele for do gnero masculino e mais jovem do que 10 anos de idade. """
    
    predictions = []
    for _, passenger in data.iterrows():
        
        if passenger['Sex'] == 'female' \
                or (passenger['Sex'] == 'male' and passenger['Age'] < 10):
            predictions.append(1)
        else:
            predictions.append(0)
    
    # Retornar nossas estimativas
    return pd.Series(predictions)

# Fazer as estimativas
predictions = predictions_2(data)

Conclusão¶

Depois de muitas iteraes explorando e condicionando os dados, voc construiu um algoritmo til para estimar a sobrevivncia de cada passageiro abordo do RMS Titanic. A tcnica aplicada neste projeto uma implementao manual de um modelo simples de machine learning, a rvore de deciso. A rvore de deciso separa um conjunto de dados em pedaos cada vez menores (chamados de ns), ao utilizar um atributo de cada vez. Cada vez que um conjunto de dados separado, nossas estimativas tornam-se mais precisas, como se cada um dos subgrupos resultantes se tornasse mais homogneo (contendo classes similares) do que antes. A vantagem de ter um computador fazendo coisas por ns que ele pode ser levado a exausto e obter resultados mais preciso do que nossa explorao manual acima. Este link fornece outra introduo em machine learning utilizando a rvore de deciso.

A rvore de deciso apenas um de muitos modelos que vem da aprendizagem supervisionada. Na aprendizagem supervisionada, ns tentamos utilizar atributos dos dados para estimar ou modelar coisas com resultados objetivos classificadas. Dito isso, cada um dos pontos dos nossos dados tem um valor de resultado conhecido, como uma categrica, discreta classe como 'Survived', ou um nmero, um valor contnuo como estimar o valor de uma casa.
